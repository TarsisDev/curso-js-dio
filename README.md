# 🚀 Formação JavaScript Developer - DIO

Este repositório foi criado para concentrar todos os desafios de código, projetos práticos e anotações que desenvolvi durante a Formação JavaScript Developer da [Digital Innovation One (DIO)](https://dio.me).

O objetivo principal é acompanhar a minha evolução lógica, registrar o aprendizado da sintaxe do JavaScript e servir como material de consulta futura.

---

## 🛠️ Tecnologias e Ferramentas Utilizadas

*   **Linguagem Principal:** JavaScript (ES6+)
*   **Ambiente de Execução:** Node.js
*   **Editor de Código:** Visual Studio Code (VS Code)
*   **Controle de Versão:** Git e GitHub

---

## 📁 Estrutura do Repositório

O projeto está organizado por módulos e tópicos para facilitar a navegação:

*   📂 `01-variaveis-operadores/` — Primeiros passos com declaração de variáveis e operadores matemáticos.
*   📂 `02-condicionais/` — Estruturas de controle de fluxo (`if`, `else`, `switch`).
*   📂 `03-funcoes/` — Declaração, escopo e reaproveitamento de blocos de código.
*   📂 `04-desafios-projetos/` — Projetos práticos completos propostos durante a formação.

---

## 👨‍💻 Como rodar os códigos localmente

Se você quiser testar qualquer um dos exercícios na sua máquina:

1. Clone o repositório:
   ```bash
   git clone https://github.com
   ```
2. Entre na pasta do exercício desejado:
   ```bash
   cd 01-variaveis-operadores
   ```
3. Execute o arquivo usando o Node.js:
   ```bash
   node "nome do arquivo".js
   ```
